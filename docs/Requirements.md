# 要求

- コミットの履歴がかわいくて見やすい感じ
- クローンが楽にできる（フォルダ内で右クリックしたときのコンテキストメニューに出てくる）
- ブランチわかりやすいといい
- 差分も簡単に見れるといい
- 画面内のメッセージはすべて日本語で表示する
- エラーメッセージ、説明文、ボタン名、確認文言も日本語に統一する

## クローン

- ユーザーはクローン元のリポジトリURLを入力できる
- ユーザーは保存先の親フォルダを選択できる
- コンテキストメニューから起動した場合、選択したフォルダのパスを初期値として表示する
- クローン時にはリポジトリ名のフォルダを保存先に作成する
- クリップボードの文字列をURLの初期値にする
